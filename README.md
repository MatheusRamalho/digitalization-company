## 💡 Projeto

O projeto é uma landing page para uma empresa que segue o ramo da digitalização de documentos, hospedando os documentos digitalizados.

<br />

## ⭐ Stack

- React
- Next
- Typescript
- Tailwind

<br />

## 🧰 Funcionalidades

## 🎯 Rodando a aplicação

Clone o repositório

```bash
git clone https://github.com/MatheusRamalho/digitization-company-web
```

Utilize o **npm** para instalar as dependências do projeto.

```bash
npm install
```

Em seguida, inicie o projeto.

```bash
npx run dev
```
